## DATOS DE LA ACTIVIDAD
Desarrollo de una aplicación Android en Kotlin que permite introducir datos personales y salariales para simular a grandes rasgos y de forma ficticia el cálculo del:
- Salario neto mensual.
- Retención de IRPF.
- Posibles deducciones.

La aplicación consta de dos pantallas:
1. Pantalla de entrada de datos (Petición de datos): permite al usuario introducir sus datos personales y salariales (edad, estado civil, número de hijos, grupo profesional, salario bruto anual, número de pagas y grado de discapacidad).
2. Pantalla de resultados: calcula el salario bruto mensual, el salario neto mensual, la retención de IRPF y deducciones aplicables.

La aplicación incluye control de teclado adecuado para los campos numéricos y navegación entre pantallas a través de `NavController`.

## OBSERVACIONES
- La aplicación se centra en la práctica del lenguaje Kotlin para el desarrollo de una aplicación. Los cálculos realizados son simplemente una simulación ficticia.
